Logo Placement Instructions
==========================

For the GuidePoint Security logo:
1. Save the GuidePoint Security logo as "guidepoint-logo.jpg" in this directory
2. Recommended size: 200px width, with transparent background
3. The logo will automatically appear in the bottom right corner of each slide

For the Jefferson Health logo:
1. Save the Jefferson Health logo as "jefferson-logo.svg" in this directory
2. Recommended size: 200px width, with transparent background
3. This can be added to the title slide by editing the index.html file

If you don't have the logos available, you can:
- Download them from the respective company websites
- Use a placeholder image temporarily
- Remove the logo references from the custom.js file and index.html
