Installation & Setup


# Extract the tool package
unzip ado-inventory-tool.zip
cd ado-inventory-tool/

# Install dependencies
pip install -r requirements.txt

# Set environment variables
export ADO_ORG="your-organization-name"
export ADO_PAT="your-personal-access-token"

Security Note: Never store your Personal Access Token in files. Always use environment variables.

Running the Tool

For each of your 7 ADO organizations:


# Basic inventory with compliance report
python ado_inventory.py --org "OrgName" --compliance-report --output-dir "./output/OrgName"

# For more detailed output
python ado_inventory.py --org "OrgName" --compliance-report --export-detailed --output-dir "./output/OrgName"

Required permissions: Code (Read), Project and Team (Read), Build (Read)
